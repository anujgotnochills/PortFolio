# **App Name**: Editorial Edge

## Core Features:

- Portfolio Showcase: Display editing work samples, categorized by niche (e.g., commercial, narrative, documentary).
- Customizable Layout: Enable users to customize the layout of their portfolio, selecting from pre-designed templates.
- Client Testimonials: Showcase testimonials from past clients to build credibility.
- Contact Form: Allow potential clients to reach out via a direct contact form.
- AI Portfolio Analyzer: Generative AI tool to provide feedback on a user's portfolio, based on their stated editing niche and desired aesthetic.

## Style Guidelines:

- Primary color: A sophisticated deep blue (#34495E) to convey professionalism and trust.
- Background color: Light gray (#F0F3F4), nearly desaturated version of the primary, to ensure readability and a clean look.
- Accent color: A muted teal (#45B39D), analogous to the primary but distinct, for interactive elements and highlights.
- Font pairing: 'Playfair' (serif) for headlines, conveying elegance, and 'PT Sans' (sans-serif) for body text, ensuring readability.
- Use a set of minimalist line icons for categories and navigation.
- A clean, grid-based layout with ample white space for a modern and presentable aesthetic.
- Subtle transitions and hover effects to add interactivity and visual appeal.
